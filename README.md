# Heart-Disease-Prediction-project
# Heart Disease Prediction using Machine Learning

## Project Overview
This project predicts whether a person has heart disease or not using Machine Learning algorithms. The project uses medical data such as age, cholesterol level, blood pressure, chest pain type, and other health-related features to make predictions.

## Objective
The main objective of this project is to build a Machine Learning model that can predict heart disease accurately using patient medical information.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Machine Learning Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Dataset
The dataset contains patient medical records with various health parameters and a target column indicating presence or absence of heart disease.

## Workflow
1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Handle Missing Values
5. Feature and Target Separation
6. Train-Test Split
7. Model Training
8. Prediction
9. Accuracy Evaluation
10. Final Disease Prediction System

## Model Evaluation
The project evaluates model performance using:
- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1-score

## Final Result
The Random Forest model achieved good prediction accuracy for heart disease detection.

## Conclusion
This project demonstrates how Machine Learning can be used in healthcare to assist in early disease prediction and data analysis.
